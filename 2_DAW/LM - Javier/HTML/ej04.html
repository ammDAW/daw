Elementos

Un elemento HTML normalmente consiste en una etiqueta de inicio y una etiqueta de fin , con el contenido insertado entre ellas.

Los elementos HTML pueden anidarse (los elementos pueden contener elementos).

No se debe olvidar de la etiqueta final. Puede producir resultados inesperados y / o errores.

Los elementos HTML sin contenido se denominan elementos vacíos. Si desea una validación más estricta, o si necesita hacer su documento legible por analizadores XML, debe cerrar todos los elementos HTML correctamente.

Utilizar etiquetas en minúsculas: Las etiquetas HTML no distinguen entre mayúsculas y minúsculas, pero se recomiendan estas y son obligatorias en XHTML.

El elemento <head>

No tiene nada que ver con los encabezados HTML

Es un conetenedor de metadatos. Los metadatos HTML son datos sobre el documeto HTML. Los metadatos no se muestran.

Los metadatos definen típicamente el título del documentos, el conjunto de caracteres, los estilos, los vínculos, los scripts y otra meta información

Meta charset como y por qué usarlo siempre

¿Por qué se debe especificar el Charset?

Aunque el meta charset no es obligatorio, se debería incluir en cualquier página web para indicar de forma expresa la codificación de caracteres utilizada y así asegurar que la página se ve correctamente. En HTML5 la codificación por defecto si no se especifica es UTF-8. No obstante, es el navegador el que finalmente decide la codificación a utilizar. Para evitar un visionado incorrecto en algunos navegadores, es recomendable incluir siempre la etiqueta <meta charset="">

Más UTF-8

A character in UTF8 can be from 1 to 4 bytes long. UTF-8 can represent any character in the Unicode standard. UTF-8 is backwards compatible with ASCII. UTF-8 is the preferred encoding for e-mail and web pages

La diferencia entre Unicode y UTF-8 Unicode es un conjunto de caracteres . UTF-8 es la codificación

En HTML, un color se puede especificar mediante el uso de un nombre de color, un valor RGB, HEX, HSL, RGBa y HSLa

En HTML, un color se puede especificar mediante el uso de un nombre de color, en este ejemplo blue <p style="color:blue">

Con RGB,ejemplo <p style="color: rgb(255,0,0)" sigue la fórmula rgb(red, green, blue), la combinación de los 3 colores básicos, rojo, verde y azul.

Los tonos de gris a menudo se definen mediante valores iguales para todas las 3 fuentes de luz <p style="color:rgb(156, 156, 156)">

Otro ejemplo de gris más oscuro, con el valor 100 en todos los colores

Un color también se puede especificar con un valor hexadecimal en la forma: #RRGGBB, donde RR (rojo), GG (verde) y BB (azul) son valores hexadecimales entre 00 y FF (igual que el decimal 0-255), en este ejemplo hemos utilizado color:#008000

HSL significa matiz, saturación y luminosidad hsl(hue, saturation, lightness) < p style="background-color:hsl(217, 32%, 80%)"> (Ver con Brackets)

El tono o matíz (hue) es un grado en la rueda de color de 0 a 360. 0 es rojo, 120 es verde y 240 es azul.
La saturación es un valor porcentual, 0% significa un tono de gris y 100% es el color completo.
La claridad, lightness () es también un porcentaje, 0% es negro, 50% no es claro ni oscuro, el 100% es blanco.
